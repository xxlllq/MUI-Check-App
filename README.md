#  :iphone: MUI-Check-App
使用MUI开发的一款用于检查打分的App，主要功能为数据采集（音频、视频、照片采集），Android与iOS真机测试。

- **首页**

首页主要用于展示分配给当前用户的检查任务，主数据为任务，点击展开则为详细检查任务子数据。支持下拉刷新，下拉加载数据列表功能。

- **检查页**

检查页主要用于音频、视频、照片等数据采集，实现的功能有调用摄像头进行拍照与查看拍照后的图片、录像与录像播放、录音以及播放音频。用户可将采集的数据上传到服务器，同时也支持用户从服务器远程查看音频（首次查看从服务器缓存在本地，以后查看直接播查看存文件）、视频、照片。
