#  :iphone: MUI-Check-App
使用MUI开发的一款用于检查打分的App，主要功能为数据采集（音频、视频、照片采集），Android与iOS真机测试。

- **检查页**

首页主要用于音频、视频、照片等数据采集，实现的功能有调用摄像头进行拍照与查看、录像与录像播放、录音以及播放音频。用户可将采集的数据上传到服务器，也支持用户从服务器远程查看音频、视频、照片。
